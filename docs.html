CricVeda API

REST API for cricket intelligence — ML-powered player predictions, Dream11 lineup optimization, and historical matchup analytics backed by 15,000+ matches of ball-by-ball data.

Base URL https://api.cricsynthesis.in/v1
Open Interactive Playground
CricVeda is currently in private beta. Request an API key from the invite form.

Authentication

Pass your API key in the X-API-Key header with every request. Never include it in query strings or URLs. 

curl https://api.cricsynthesis.in/v1/matches/upcoming \
  -H "X-API-Key: cv_live_your_key_here"
import requests

res = requests.get(
    "https://api.cricsynthesis.in/v1/matches/upcoming",
    headers={"X-API-Key": "cv_live_your_key_here"},
)
print(res.json())
const res = await fetch(
  "https://api.cricsynthesis.in/v1/matches/upcoming",
  { headers: { "X-API-Key": "cv_live_your_key_here" } }
);
const data = await res.json();
Keep your API key secret. If it's compromised, contact support to rotate it.

Rate Limits

Free tier allows 100 requests per day per API key. Response headers tell you your current usage:

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 87
X-RateLimit-Reset: 1735689600

Responses are cached (matches 5 min · predictions 1 hour · player form 30 min), so repeated calls within the cache window don't consume quota.

Error Responses

StatusMeaning
401Missing X-API-Key header
403Invalid API key
404Resource not found (match / player)
422Validation error — check request params
429Rate limit exceeded
500Internal server error

Upcoming Matches

GET /v1/matches/upcoming List scheduled fixtures

Returns upcoming T20 and ODI fixtures ordered by match date. Cached for 5 minutes.

Query parameters

NameTypeRequiredDescription
league_id string optional Filter by league (e.g. ipl, t20i, odi)
format string optional T20 or ODI
limit integer optional Max results (1–100, default 20)

Example request

curl "https://api.cricsynthesis.in/v1/matches/upcoming?league_id=ipl&limit=5" \
  -H "X-API-Key: cv_live_your_key_here"
res = requests.get(
    "https://api.cricsynthesis.in/v1/matches/upcoming",
    params={"league_id": "ipl", "limit": 5},
    headers={"X-API-Key": API_KEY},
)

Example response

[
  {
    "upcoming_id": 42,
    "league_id": "ipl",
    "match_date": "2026-03-22",
    "team1": "Mumbai Indians",
    "team2": "Chennai Super Kings",
    "format": "T20",
    "status": "scheduled"
  }
]

Returns XGBoost-predicted Dream11 fantasy points for every player in both squads, sorted by predicted score. Cached for 1 hour after lineup confirmation.

NameTypeDescription
upcoming_idintegerID from /matches/upcoming

GET /v1/matches/{upcoming_id}/dream-team Optimal Dream11 lineup

Runs a linear program (PuLP) over predicted scores and returns the optimal 11-player Dream11 lineup with captain and vice-captain picks.

Constraints applied

RuleValue
Total credits≤ 100
WK slots1 – 4
BAT slots3 – 6
AR slots1 – 4
BOWL slots3 – 6
Max from one team7

Player Analytics

GET /v1/players/{player_id}/form Rolling form, trend, and recent scores

Returns a player's current form profile — rolling fantasy point averages, batting and bowling stats, and their last 5 match scores. Cached for 30 minutes.

Example response

{
  "player_id": 253802,
  "name": "Virat Kohli",
  "role": "BAT",
  "fp_last3_avg": 61.2,
  "fp_last5_avg": 54.8,
  "fp_trend": "rising",
  "bat_runs_avg5": 44.6,
  "bat_sr_avg5": 142.3,
  "recent_scores": [
    { "match_date": "2026-03-18", "total_points": 76.5 },
    { "match_date": "2026-03-14", "total_points": 48.0 }
